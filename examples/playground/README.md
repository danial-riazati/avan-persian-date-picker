# Avan Playground

Local browser demo for **Avan Persian Date Picker**.

**Live demo:** [danialriazati.ir/avan-persian-date-picker](https://danialriazati.ir/avan-persian-date-picker/)

## Run locally

```bash
corepack pnpm dev:playground
```

Open [http://localhost:5173](http://localhost:5173).

## GitHub Pages (main branch only)

CI builds the app and commits it to **`playground/`** on `main` (auto-generated — do not edit by hand).

**Settings → Pages → Deploy from branch → `main` → `/playground`**
